## Startup & Commercial Capital Intelligence System

This project is a research-driven fintech backend designed to evaluate
startup and MSME capital readiness.

### Research Foundations
- Commercial credit underwriting
- Cash-flow based lending
- Logistic regression for PD estimation
- Policy-driven capital suitability
- LLM-based explainability (Gemini 2.5)

### AI Usage
Machine Learning estimates risk.
Gemini explains decisions and suggests schemes.
AI never makes financial decisions.
